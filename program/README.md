# Dijkstra最短路径算法 - Qt应用程序

这是一个基于Qt框架实现的Dijkstra最短路径算法应用程序，适用于Mac平台。

## 功能特性

1. **文件加载**：支持从文本文件加载节点和距离数据
2. **手动添加**：可以手动添加节点之间的关系和距离
3. **路径计算**：计算两个节点之间的最短路径
4. **结果显示**：显示最短路径、距离和路径节点

## 数据格式

数据文件应为文本格式，每行包含三个数字，用空格、制表符、逗号或分号分隔：

```
节点1ID  节点2ID  距离值
```

示例：
```
101	102	6
101	103	3
102	103	2
102	104	5
103	104	3
```

## 编译和运行

### 前提条件

- Qt 5.12 或更高版本
- C++17 编译器
- macOS 开发环境

### 编译步骤

1. 打开终端，进入程序目录：
```bash
cd "程序"
```

2. 使用qmake生成Makefile：
```bash
qmake DijkstraApp.pro
```

3. 编译：
```bash
make
```

或者使用Qt Creator：
1. 打开Qt Creator
2. 选择"打开项目"
3. 选择 `DijkstraApp.pro` 文件
4. 点击"构建"按钮

### 运行

编译完成后，可执行文件位于：
- Debug模式：`程序/DijkstraApp.app/Contents/MacOS/DijkstraApp`
- Release模式：`程序/DijkstraApp.app/Contents/MacOS/DijkstraApp`

或者直接在Qt Creator中点击"运行"按钮。

## 使用说明

1. **加载数据文件**：
   - 点击"加载数据文件"按钮
   - 选择数据文件（默认会打开 `Dijkstra算法` 文件夹）
   - 成功加载后，状态栏会显示节点数量

2. **手动添加节点**：
   - 在"手动添加节点关系"区域输入：
     - 节点1 ID
     - 节点2 ID
     - 距离值
   - 点击"添加"按钮

3. **计算最短路径**：
   - 在"计算最短路径"区域输入：
     - 起始节点ID
     - 终止节点ID
   - 点击"计算"按钮
   - 结果会显示在"计算结果"区域

4. **清空数据**：
   - 点击"清空数据"按钮可以清除所有已加载的数据

## 算法说明

本程序实现了经典的Dijkstra最短路径算法：
- 支持无向图
- 支持非负权值
- 支持多条最短路径（选择其中一条）
- 使用邻接表存储图结构

## 注意事项

- 距离值必须为正数
- 节点ID可以是任意整数
- 如果两个节点之间存在多条边，距离值必须相同
- 如果两个节点之间无路径，会显示"无路径可达"
